Rotten fruit classifier using CNN

AS WE ALL KNOW IT IS HARD TO BUY ONLY GOOD FRUITS WHEN WE GO TO SHOPPING. IT IS NOT ONLY TIME CONSUMING BUT SOMETIMES IT'S BECOME DIFFICULT TO IDENTIFY THE GOOD ONE FROM BAD ROTTEN ONE BECAUSE OF LIGHTING CONDITION AND POEPLES. SO WE CREATED A CLASSIFIER WHICH CAN PREDICT A GOOD ONE FROM BAD ONE WITH THE ACCURACY OF 89%. IT ALSO ABLE TO SAY THE WORST CONDITION AT 49% AND THE BEST CONDITION OF ROTTEN FRUIT ABOUT 68%. WE USED CNN, TENSORFLOWM, KERAS AND NUMPY IN THIS RESEARCH PROJECT.
